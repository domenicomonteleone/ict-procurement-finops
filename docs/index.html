Featured project

FinOps Budget Intelligence System

Automated FinOps reporting system developed to support ICT procurement decisions in an enterprise cloud environment, without relying on expensive SaaS tools. Built on top of passive invoice data, the system classifies spend by vendor, category (OPEX/CAPEX) and recurrence type — enabling continuous monitoring of budget consumption, supplier behaviour and cost anomalies across multiple ICT categories and a structured vendor panel. Outputs a multi-sheet Excel report with spend trends, year-over-year variance, budget vs. forecast and automated alerts for significant cost deviations.

Python SQLite Excel Report FinOps · Spend Analysis · Vendor Tracking

Vendor Classification Pipeline
